Seller Data Dashboard (Excel Project)

🔹 Project Overview

This project is an interactive Seller Data Dashboard created using Microsoft Excel. It helps analyze sales performance based on different categories such as city, product items, and time.


---

🛠 Tools Used

Microsoft Excel

Pivot Tables

Charts

Slicers & Timeline



---

📈 Features

Total Sales Amount KPI

City-wise Sales Analysis

Month-wise Sales Trend

Quantity of Items Sold (Pie Chart)

Interactive Slicers (Filter by Item & City)

Timeline Filter for Date Analysis
